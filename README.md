# Movies App

### Installation

1. Clone the repository:

```shell
git clone https://github.com/WilliamsMata/movies-app.git
```

2. Navigate to the project directory:

```shell
cd movies-app
```

3. Install the dependencies:

```shell
npm install
```

4. Add env:

```shell
cp ./env.template ./.env
```

### Usage

1. Start the development server:

```shell
npm start
```
